# Finance Analyzer Dashboard (Frontend)

React frontend for a personal finance analytics dashboard.

## Features
- Upload transaction CSV files
- View total spend, stats, and anomalies
- Explore category and monthly trend charts
- View prediction insights when available

## Tech Stack
- React 18
- React Scripts (Create React App)
- Recharts
- Axios

## Getting Started

### Prerequisites
- Node.js 18+ (recommended)
- npm

### Install
```bash
npm install
```

### Run locally
```bash
npm start
```
The app runs at `http://localhost:3000`.

## Available Scripts
- `npm start` - start development server
- `npm run build` - create production build
- `npm test` - run tests

## Backend API
